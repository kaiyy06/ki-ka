# Gallery Images

Place your couple/memory photos here. They will appear in the 3D carousel gallery.

**Recommended:** 15 images, portrait orientation (3:4 ratio), ~500px wide.

Name them sequentially:
- `1.jpg`
- `2.jpg`
- `3.jpg`
- ... up to `15.jpg`

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
